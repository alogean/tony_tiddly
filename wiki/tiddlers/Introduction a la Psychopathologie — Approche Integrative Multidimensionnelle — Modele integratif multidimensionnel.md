# Modèle intégratif multidimensionnel

> Section de [[intro_psychopathologie]]

Le modèle intégratif combine toutes les dimensions :

```
Influences biologiques (génétique, neurobiologie)
        ↕
Influences psychologiques (cognition, comportement, émotion)
        ↕
Influences sociales (culture, relations, contexte)
        ↕
Influences développementales (trajectoire de vie)
```

**Principe clé** : ces dimensions interagissent de manière **bidirectionnelle** et **dynamique** — chacune influence et est influencée par les autres.
