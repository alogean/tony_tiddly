# Diagnostic différentiel — Méthode Sherlock Holmes

> Section de [[marche_a_suivre_etude_cas]]

### Principe
Lister **plusieurs hypothèses diagnostiques** plausibles, argumenter **pour** et **contre** chacune, éliminer progressivement les moins cohérentes. Basé sur un **faisceau d'indices concordants**.

### Structure recommandée

1. **Phrase introductive** : annoncer les **familles de troubles** explorées
   - Exemple : *« Au vu du tableau sémiologique et étant donné les connaissances actuelles sur le cours, nous pouvons retenir plusieurs orientations diagnostiques : 1) Troubles anxieux 2) Troubles liés à des facteurs de stress »*
   - ⚠️ Rester **général**, ne pas être trop spécifique d'emblée

2. **Organisation par familles de troubles** :
   - Regrouper les troubles apparentés sous un même axe
   - Pour chaque famille, citer plusieurs troubles pertinents
   - Pour chaque trouble : argument **pour** + argument **contre**

3. **Ne pas citer** les critères A, B, C du DSM lettre par lettre

4. **Garder le candidat préférentiel pour la fin** (réserve dramatique)

### Exemple de structure
- *Troubles anxieux* : phobie spécifique, anxiété sociale, TAG, etc.
  - Pour chacun : argument pro + contre
- *Troubles liés au stress* : TSPT, trouble de stress aigu, trouble de l'adaptation
  - Pour chacun : argument pro + contre
- *Troubles psychotiques* : à écarter rapidement si pas de symptômes cardinaux

### Toujours exclure en premier
- Trouble dû à une **affection médicale** → examen somatique
- Trouble induit par une **substance** → bilan toxicologique
