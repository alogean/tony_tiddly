# Éléments théoriques du cours

> Section de [[marche_a_suivre_etude_cas]]

Montrer la maîtrise du **contenu théorique du module M17** en liant le cas à des concepts du cours :

### Modèles transversaux (toujours mobilisables)
- **Modèle intégratif multidimensionnel** (Barlow) : biologique + psychologique + social + développemental
- **Modèle diathèse-stress** : prédisposition + stresseurs - ressources = seuil pathogène (cf. `concepts_cles/modele_diathese_stress_laborit.md`)
- **Modèle 3D + 4D** : Déviance + Dysfonction + Détresse + Inflexibilité (cf. `concepts_cles/modele_3D_4D.md`)
- **Approche biopsychosociale**

### Modèles spécifiques au trouble
| Trouble | Modèles à mobiliser |
|---------|--------------------|
| Troubles anxieux | Triple vulnérabilité (Barlow), modèle de Borkovec (TAG), modèle de Clark (panique) |
| Troubles psychotiques | Hypothèse dopaminergique, modèle de Frith, jumelles Genain |
| TSPT | Conditionnement de la peur, hyperactivation, 3 syndromes (reviviscence/évitement/hyperactivation) |
| Dépression | Modèle cognitif de Beck, impuissance acquise (Seligman), rumination (Nolen-Hoeksema) |
| Insomnie | Modèle 3P de Spielman (prédisposants/précipitants/perpétuants) |
| TOC | Modèle cognitivo-comportemental, théorie de l'inflation des responsabilités |

### Données de base à intégrer
- **Épidémiologie** : prévalence, ratio H/F, âge de début
- **Évolution du DSM** : changements nosographiques pertinents
- **Sémiologie** : démontrer la maîtrise du vocabulaire technique
- **Comorbidités** fréquentes
