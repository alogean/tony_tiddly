# ⚠️ Règle d'or préalable — L'ordre d'exclusion

> Section de [[marche_a_suivre_etude_cas]]

Avant TOUT diagnostic psychiatrique :

```
1. Exclure les causes ORGANIQUES (affection médicale)
2. Exclure les SUBSTANCES (intoxication, sevrage, médicament)
3. → Seulement ensuite : diagnostic psychiatrique
```

> Putois (Session 3) : *« Je vais être très, très chiant avec vous : posez TOUJOURS la question de l'affection médicale et des substances. SYSTÉMATIQUEMENT. Même si le cas semble évident. Même si vous savez que vous allez réfuter ces hypothèses. C'est pour avoir les bons réflexes. »*

### Pièges organiques classiques (à connaître)
- **Hyperthyroïdie** → trouble anxieux (précurseur de l'adrénaline)
- **Carence en fer** (ferritine basse) → syndrome des jambes sans repos → insomnie chronique (30 ans de somnifères inutiles !)
- **Levothyrox mal dosé** → anxiété
- **LSD** → hallucinations (pas un trouble psychotique)
- **Hypothyroïdie** → tableau dépressif
- **Hypoglycémie** → angoisse, confusion
- **Tumeur frontale** → désinhibition, troubles cognitifs
