# Points clés pour les révisions

> Section de [[semiologie_psychopathologique_complete]]

1. Distinguer **signe** (objectif) vs **symptôme** (subjectif)
2. La sémiologie **décrit**, n'explique pas
3. Le lexique sémiologique = outil de base du **statut mental**
4. **Parathymie** = discordance émotion/contexte → désorganisation (schizophrénie)
5. **Barrage ≠ Fading** : interruption brutale vs ralentissement progressif
6. **Hallucination psychique vs vraie** : sans vs avec support sensoriel
7. **Dépersonnalisation = soi** ; **déréalisation = monde**
8. **Insight** = facteur pronostique majeur
9. **Hallucinations auditives** = les plus fréquentes en schizophrénie
10. Au relevé sémiologique : forme **narrative**, vocabulaire **technique**, pas de liste à puces
