# Points clés à retenir pour les révisions

> Section de [[troubles_psychotiques]]

1. **Bleuler** a inventé "schizophrénie" — scission associative, pas personnalités multiples
2. **Symptômes positifs** (excès) vs **négatifs** (déficits) vs **désorganisation**
3. **Hallucinations auditives** = les plus fréquentes → activation de Broca
4. **Critères DSM-5 schizophrénie** : ≥2 symptômes (≥1 parmi délires/hallucinations/discours désorganisé) pendant ≥1 mois, durée totale ≥6 mois
5. **Trouble schizophréniforme** = schizophrénie mais 1-6 mois
6. **Trouble schizo-affectif** = schizophrénie + épisode thymique simultané
7. **Jumelles Genain** : même génome, sévérités différentes → gènes + environnement
8. **Hypothèse dopaminergique** : excès dopamine → symptômes positifs ; antipsychotiques bloquent D2
9. **Modèle de Frith** : déficit de métareprésentation
10. **Catatonie** = diagnostic indépendant dans le DSM-5
11. Schizophrénie ≠ TDI (trouble dissociatif de l'identité)
