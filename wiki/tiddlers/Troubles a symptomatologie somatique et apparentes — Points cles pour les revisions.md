# Points clés pour les révisions

> Section de [[troubles_somatiques]]

1. Le DSM-5 a **renommé** "troubles somatoformes" → "troubles à symptomatologie somatique"
2. Le focus n'est plus sur "absence d'explication médicale" mais sur **la réaction excessive**
3. **Hypocondrie disparue** → scindée en : trouble à symptomatologie somatique (avec symptômes) ou crainte excessive d'avoir une maladie (sans symptômes)
4. La **conversion** est inconsciente ; le trouble **factice** est conscient mais sans gain externe ; la **simulation** est consciente avec gain externe
5. **Trouble factice imposé à autrui** : c'est l'auteur qui reçoit le diagnostic, pas la victime
6. Toujours penser au **bénéfice secondaire** dans le diagnostic différentiel
