# Points clés pour les révisions

> Section de [[troubles_humeur]]

1. **3A** de la dépression : Asthénie, Aboulie, Anhédonie
2. Dépressif caractérisé = **2 semaines**, dysthymie = **2 ans**
3. **Double dépression** = dysthymie + épisode dépressif superposé
4. Bipolaire 1 = **maniaque** suffit, Bipolaire 2 = **hypomaniaque + dépressif** obligatoires
5. Maniaque vs hypomaniaque : la frontière c'est l'**hospitalisation** et l'**absence de psychose**
6. Cyclothymie = version atténuée et chronique du bipolaire
