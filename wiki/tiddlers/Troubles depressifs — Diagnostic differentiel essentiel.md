# Diagnostic différentiel essentiel

> Section de [[troubles_depressifs]]

| À écarter | Argument clé |
|-----------|--------------|
| **Trouble bipolaire** | Antécédent ou présence d'épisode maniaque/hypomaniaque |
| **Deuil normal** | Réaction culturellement attendue, fluctuante (DSM-5 a supprimé l'exclusion du deuil — diagnostiquer EDM si critères remplis) |
| **Trouble de l'adaptation** | Stresseur identifiable + symptômes < seuil EDM |
| **Trouble dépressif induit par substance** | Lien temporel + résolution à l'arrêt |
| **Hypothyroïdie** | Bilan thyroïdien systématique (Putois insiste : organique d'abord !) |
| **Démence débutante** | Pseudo-démence dépressive vs vraie démence (évaluation cognitive) |
| **Trouble dépressif dû à affection médicale** | Lien physiopathologique direct |
