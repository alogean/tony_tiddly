# Points clés pour les révisions

> Section de [[troubles_depressifs]]

1. **3A de Putois** : Asthénie + Aboulie + Anhédonie
2. **Critères EDM** : ≥ 5 symptômes / 2 semaines, dont humeur OU anhédonie obligatoire
3. **Dysthymie** : ≥ 2 ans + ≥ 2 symptômes — chronicité > intensité
4. **Double dépression** = dysthymie + EDM superposé
5. **DMDD** = nouveau trouble pédiatrique (DSM-5) pour réduire surdiagnostic bipolaire enfant
6. **TDPM** : confirmation prospective sur ≥ 2 cycles **obligatoire**
7. **Mélancolique** vs **atypique** : opposés sur appétit, sommeil, réactivité
8. **Modèle de Beck** : triade cognitive (soi/monde/futur)
9. **DSM-5 a supprimé l'exclusion du deuil** — controverse majeure (Frances et al.)
10. **Marcher 1h/jour** ≈ antidépresseur (Putois)
11. **ISRS** = première ligne pharmaco ; **TCC** ou **IPT** = premières lignes psy
12. **Toujours exclure** hypothyroïdie, anémie, carences, substances avant diagnostic
