# Trouble dépressif caractérisé — 296.xx (F32.x ou F33.x)

> Section de [[troubles_depressifs]]

**Diagnostic** : ≥ 1 épisode dépressif caractérisé, **jamais** d'épisode maniaque/hypomaniaque.

**Codage** :
- F32.x = épisode unique
- F33.x = épisodes récurrents

### Spécificateurs (très utilisés à l'examen)

| Spécificateur | Caractéristiques |
|---------------|-----------------|
| **Avec détresse anxieuse** | Tension, agitation, peur — souvent associé à un risque suicidaire ↑ |
| **Avec caractéristiques mixtes** | ≥ 3 symptômes maniaques/hypomaniaques pendant l'épisode dépressif |
| **Avec caractéristiques mélancoliques** | Anhédonie sévère, absence de réactivité affective, **dépression pire le matin**, **réveils précoces**, ralentissement psychomoteur, anorexie, culpabilité massive |
| **Avec caractéristiques atypiques** | **Réactivité de l'humeur**, hyperphagie, hypersomnie, **paralysie de plomb** (lourdeur des membres), sensibilité au rejet |
| **Avec caractéristiques psychotiques** | Délires/hallucinations, **congruents** (thèmes dépressifs : ruine, faute, mort, néant) ou **non congruents** |
| **Avec catatonie** | Mutisme, immobilité, flexibilité cireuse |
| **Avec début péri-partum** | Pendant grossesse ou ≤ 4 semaines post-accouchement |
| **Avec caractère saisonnier** | Relation temporelle constante avec saisons (≥ 2 ans) — typiquement automne/hiver |

### Évolution
- Épisode unique vs récurrent
- Sévérité : léger / moyen / grave
- Avec/sans psychose
- En rémission partielle / complète

### Épidémiologie
- Prévalence sur 12 mois : ~7%
- Prévalence vie entière : 16-20%
- Ratio F:H = 1,5 à 3:1
- Pic de début : 20-30 ans (mais possible à tout âge)
- Risque suicidaire significatif
