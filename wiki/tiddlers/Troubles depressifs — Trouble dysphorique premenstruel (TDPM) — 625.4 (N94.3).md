# Trouble dysphorique prémenstruel (TDPM) — 625.4 (N94.3)

> Section de [[troubles_depressifs]]

**Critère A** : Dans la **majorité des cycles**, ≥ 5 symptômes dans la **semaine précédant les règles**, s'améliorant dans les jours suivant le début, minimaux/absents la semaine après.

**Critère B** : ≥ 1 parmi (symptômes thymiques) :
1. Labilité émotionnelle marquée
2. Irritabilité/colère/conflits accrus
3. Humeur dépressive, désespoir, idées dévalorisantes
4. Anxiété, tension, "à cran"

**Critère C** : ≥ 1 parmi (autres symptômes) — somme des B+C ≥ 5 :
- Perte d'intérêt
- Difficulté de concentration
- Léthargie/fatigue
- Modification appétit/cravings
- Hypersomnie/insomnie
- Sensation d'être dépassée
- Symptômes physiques (sensibilité mammaire, ballonnement, douleur articulaire/musculaire, prise de poids)

**Critère F** : **Confirmation prospective** sur ≥ 2 cycles (échelles validées) — **règle d'or diagnostique**.

⚠️ Diagnostic souvent surévalué cliniquement — la confirmation prospective est essentielle pour le distinguer du syndrome prémenstruel commun.
