# Troubles de la personnalité

**Dernière mise à jour** : 2026-05-14
**Sources** : Barlow & Durand (2016), *Psychopathologie - Une Approche Intégrative*, ch. 12 (pp. 440-472) ; DSM-5

---

## Définition générale (Barlow p. 441)

Mode **durable** d'émotions, cognitions et comportements qui :
- entraîne une détresse chez la personne et/ou son entourage
- cause des difficultés sociales/professionnelles
- est **chronique** (depuis l'enfance/adolescence, tout au long de la vie adulte)
- est **inadapté** et **rigide**

### Critères DSM-5 du « Trouble général de la personnalité »
- **A.** Déviation notable des attentes culturelles dans ≥ 2 domaines : cognition, affectivité, fonctionnement interpersonnel, contrôle des impulsions
- **B.** Rigide et envahissant des situations diverses
- **C.** Souffrance/altération significative
- **D.** Stable, débute à l'adolescence ou début âge adulte
- **E.** Non expliqué par autre trouble mental
- **F.** Non imputable à substance/affection médicale

## Les 3 groupes (clusters) DSM-5

### Groupe A — Bizarre/excentrique
| Trouble | Caractéristique centrale |
|---|---|
| **Paranoïaque** | Méfiance soupçonneuse envahissante |
| **Schizoïde** | Détachement social + restriction émotionnelle |
| **Schizotypique** | Déficits sociaux + distorsions cognitives/perceptuelles + excentricité |

### Groupe B — Théâtral/émotif/capricieux
| Trouble | Caractéristique centrale |
|---|---|
| **Antisociale** | Mépris et violation des droits d'autrui |
| **Borderline (limite)** | Instabilité (relations, image de soi, affects) + impulsivité |
| **Histrionique** | Émotivité excessive + quête d'attention |
| **Narcissique** | Grandiosité + besoin d'admiration + manque d'empathie |

### Groupe C — Anxieux/craintif
| Trouble | Caractéristique centrale |
|---|---|
| **Évitante** | Inhibition sociale + sentiment d'inadéquation + peur du rejet |
| **Dépendante** | Besoin excessif d'être pris en charge + peur séparation |
| **Obsessionnelle-compulsive** | Ordre/perfectionnisme/contrôle (≠ TOC) |

## Fiches synthétiques par trouble

### Groupe A

#### Paranoïaque (p. 446-449)
- **Clinique** : méfiance injustifiée, soupçonne intentions malveillantes, difficile à se confier, rancunier, ergoteur, hypersensible critique, besoin autonomie
- **Causes** : faible contribution biologique (lien faible avec schizophrénie) ; schémas erronés sur autrui ; maltraitance précoce ; facteurs culturels (immigrants, prisonniers, malentendants)
- **Traitement** : très difficile (méfiance envers thérapeute) ; approche cognitive (modifier schémas) ; alliance thérapeutique = 1er objectif ; pas de démonstration d'efficacité

#### Schizoïde (p. 449-450)
- **Clinique** : détachement social, gamme restreinte d'émotions, préfère solitude, froid, indifférent éloges/critiques, pas de désir relations proches
- **Causes** : peu de données ; faibles récepteurs dopaminergiques ; timidité infantile précurseur ; maltraitance/négligence ; lien avec autisme suggéré
- **Traitement** : rarement consultent ; sensibilisation aux émotions, jeu de rôles, entraînement empathie et aptitudes sociales ; pronostic limité

#### Schizotypique (p. 451-452)
- **Clinique** : isolement + croyances/comportements bizarres ; idées de référence, pensée magique, perceptions inhabituelles, langage bizarre, méfiance, affects inadéquats, apparence excentrique
- **Lien continuum schizophrénie** : phénotype moins sévère du génotype schizophrénique (Meehl)
- **Causes** : forte composante génétique (familles schizophrènes), maltraitance infantile, anomalies cérébrales (hémisphère gauche)
- **Traitement** : antipsychotiques + TCC + soutien communautaire ; **stratégie préventive** pour retarder évolution vers schizophrénie ; 30-50% comorbidité dépressive

### Groupe B

#### Antisociale (p. 453-460)
- **Clinique** : violation droits d'autrui depuis 15 ans (TC préalable) ; tromperie, impulsivité, agressivité, irresponsabilité, absence remords
- **Psychopathie (Hare, PCL-R)** ≠ DSM antisociale :
  - DSM = comportements observables (plus fiable)
  - Psychopathie = traits de personnalité (charme superficiel, manipulation, manque empathie)
- **Causes** :
  - Génétiques (interaction gène-environnement, Crowe)
  - Neurobiologiques :
    - **Hypothèse sous-stimulation** (Yerkes-Dodson) : seuil cortical bas → recherche stimulation
    - **Hypothèse absence de peur** (Lykken) : seuil de peur élevé, déficit amygdalien
    - Modèle Gray : déséquilibre SIC/SR (inhibition < récompense)
  - Développementales : TC précoce, mécanisme familial coercitif (Patterson), milieu défavorisé
- **Évolution** : s'atténue après 40 ans
- **Traitement** : très pessimiste ; **prévention précoce** > traitement adulte ; TCC chez délinquants violents

#### Borderline (p. 461-464)
- **Clinique** : instabilité (relations, image de soi, affects) + impulsivité ; abandon évité ; idéalisation/dévalorisation ; automutilations ; tentatives suicide (6% décès) ; vide chronique ; colère ; symptômes dissociatifs transitoires
- **Comorbidités** : dépression (20%), bipolaire (40%), substances (>67%), boulimie (25%)
- **Causes (triple vulnérabilité Barlow)** :
  1. Vulnérabilité biologique généralisée (réactivité émotionnelle, sérotonine)
  2. Vulnérabilité psychologique généralisée (monde menaçant)
  3. Vulnérabilité spécifique apprise (**abus précoces** sexuels/physiques, négligence)
- Système limbique (amygdale, hippocampe) ; biais sensibilité honte ; biais mémoire mots négatifs
- **Traitement** :
  - **TCD (Thérapie Comportementale Dialectique) de Marsha Linehan** = gold standard
  - Hiérarchie : suicidaires → interférences thérapie → qualité de vie
  - Validation, régulation émotionnelle, exposition style PTSD
  - Médication adjuvante : stabilisateurs humeur, antipsychotiques
- **Évolution** : amélioration après 30 ans ; 88% rémission à 10 ans

#### Histrionique (p. 464-466)
- **Clinique** : théâtralité, quête attention, séduction inappropriée, émotions superficielles changeantes, parole vague émotionnelle, suggestibilité, surévalue intimité
- **Causes** : très peu recherchées ; co-occurrence forte avec antisociale → hypothèse étiologie commune typée par genre (Lilienfeld) ; biais culturels féminins (Kaplan)
- **Traitement** : peu de réussites documentées ; centré sur problèmes interpersonnels, montrer coûts à long terme, renforcement comportements adaptés

#### Narcissique (p. 466-467)
- **Clinique** : grandiosité, besoin admiration, sentiment d'être spécial, fantasmes succès illimité, exploitation, manque empathie, envie, arrogance
- **Vulnérabilité dépression** quand réalité < attentes
- **Causes** :
  - **Kohut (psychodynamique)** : échec parental à inculquer empathie → fixation centralité, quête parent idéal
  - **Lasch (sociologique)** : « culture narcissique » occidentale, hédonisme, individualisme ; ↑ chez Baby Boomers
- **Traitement** : recherche très limitée ; Beck (cognitif) cible mégalomanie + hypersensibilité critique + manque empathie ; remplacer grandiosité par plaisirs réalistes ; traitement dépression à l'âge mûr

### Groupe C

#### Évitante (p. 468-469)
- **Clinique** : hypersensibilité jugement négatif, inhibition sociale, sentiment inadéquation, faible estime soi, peur rejet/ridicule
- **Différence clé schizoïde** : **désire les relations** mais évite par anxiété (schizoïde ne désire pas)
- **Proximité forte avec phobie sociale**
- **Causes** : tempérament prédisposant + **rejet parental précoce**, manque affection (Stravynski) ; sous-schizophrénie débattue (Fogelson)
- **Traitement** : **nombreuses études contrôlées** (rare pour TP) ; TCC, exposition graduelle, désensibilisation systématique, entraînement compétences sociales ; alliance thérapeutique = forte prédictrice

#### Dépendante (p. 469)
- **Clinique** : besoin pris en charge, soumis/collant, peur séparation, difficile décider seul, ne pas exprimer désaccord, cherche urgemment nouvelle relation à la fin d'une précédente
- **Ressemble évitant** mais s'accroche au lieu d'éviter
- **Causes** : contributions génétiques probables ; **perte/rejet parental précoce**, négligence, mort parent (Stone)
- **Considération culturelle** : confucianisme asiatique (soumission valorisée) → prudence diagnostic
- **Traitement** : apparaissent « patients idéaux » mais cela contredit le but ; progression graduelle vers autonomie ; éviter dépendance au thérapeute

#### Obsessionnelle-compulsive (p. 470-471)
- **Clinique** : fixation « seule bonne façon de faire » ; ordre, perfectionnisme, contrôle ; perfectionnisme entrave achèvement ; dévotion excessive travail ; conscience morale rigide ; incapacité jeter, déléguer ; avarice ; entêtement
- **⚠ DIFFÉRENT DU TOC** : pas d'obsessions/compulsions au sens TOC, centré sur traits personnalité
- **Causes** : faible contribution génétique ; renforcement parental conformité normes rigides ; lien possible avec tueurs en série/pédophiles (besoin contrôle)
- **Traitement** : TCC type TOC ; cibler peurs sous-jacentes ; relaxation et distraction pour compulsions de pensée

## Concepts transversaux importants

### Modèles catégoriels vs dimensionnels
- **Catégoriel** (DSM-5 actuel) : oui/non, simplification clinique mais réification
- **Dimensionnel** : continuums (ex. **Big Five / OCEAN** : extraversion, amabilité, conscience, neuroticisme, ouverture)
- DSM-5 inclut un modèle alternatif dimensionnel en section III (« modèles émergents »)

### Comorbidité
- **Très forte chevauchement** entre TP (Tableau 12.4)
- Borderline ⇄ schizotypique (15,2x) ; borderline ⇄ paranoïaque (12,3x) ; antisociale ⇄ narcissique (14x)
- Soulève question : catégories trop fines ?

### Biais sexuels diagnostiques
- Étude Ford & Widiger (1989) : mêmes symptômes diagnostiqués différemment selon le genre
  - Symptômes antisociaux : diagnostic antisocial si homme, **histrionique si femme**
- Critères eux-mêmes potentiellement biaisés (stéréotypes féminins → histrionique)

### Considérations culturelles
- Pratiques religieuses (rituels Vaudou, chamanisme) peuvent ressembler à schizotypique
- Soumission valorisée dans confucianisme asiatique peut ressembler à dépendant
- Importance de la sensibilité interculturelle dans le diagnostic
